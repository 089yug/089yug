下午茶的搭配原則

下午茶講究茶與甜點之間的平衡，透過苦、甜與香氣的搭配， 讓整體風味更有層次。以下為常見且實用的下午茶搭配原則。

🍰 甜度平衡

甜點的甜度不宜高於茶飲的風味強度，過甜的甜點容易掩蓋茶的香氣， 使整體風味失去平衡。清香型茶飲適合搭配低甜度甜點， 而風味較濃的茶則能搭配奶油或巧克力類甜點。

🍃 風味濃淡對應

茶與甜點的風味強度應相近，才能互相襯托。 風味清淡的茶適合口味單純的甜點， 而風味厚實的茶則能搭配層次較豐富的甜點。

☕ 口感互補

茶飲的清爽口感可平衡甜點的油膩感，使整體品嚐體驗更加舒適。 奶油或起司類甜點適合搭配紅茶或烏龍茶， 酥脆甜點則適合搭配口感滑順的茶飲。

🌼 香氣協調原則

茶與甜點若在香氣上能互相呼應，能提升整體風味的一致性。 例如帶有柑橘香氣的伯爵茶，適合搭配檸檬或水果風味的甜點。

🌡️ 溫度對比

茶飲與甜點的冷熱搭配能增加味覺層次。 熱茶適合搭配冷藏或常溫甜點， 冰茶則適合搭配常溫蛋糕或餅乾。

⚠️ 避免搶味

過於強烈的香料或酒味甜點，容易掩蓋茶飲原本的風味， 搭配時應避免讓單一風味過於突出。

🌿 為什麼需要搭配指南？

很多人喜歡下午茶，卻不知道茶與甜點其實有搭配原則。 SUM 下午茶希望用簡單的方式， 帶你從茶的風味出發，找到最適合自己的甜點組合， 讓每一次下午茶都更好喝、也更有質感。

紅茶 Black Tea

簡介

產地： 主要來自印度、大吉嶺、阿薩姆、斯里蘭卡及部分台灣地區。

香氣 / 味道： 帶有焦糖、麥芽香，口感溫潤，回甘帶甜。

特色： 茶葉全發酵，茶色紅亮，是全球飲用最廣泛的茶類。

沖泡建議

水溫： 90–95°C

茶水比例： 1g 茶葉 : 50ml 水（個人口味可調整）

浸泡時間： 2–3 分鐘

※ 想喝更濃郁可延長時間，但避免過久造成苦澀

適合搭配的茶點

紅茶適合搭配：

  • 司康
  • 奶油餅乾
  • 檸檬蛋糕
  • 起司甜點

綠茶 Green Tea

簡介

產地： 中國、日本、台灣

香氣 / 味道： 清香、草本味，口感清爽

特色： 未發酵，茶色綠亮，回甘明顯，適合清淡口味。

沖泡建議

水溫： 75–85°C

茶水比例： 1g 茶葉 : 50ml 水

浸泡時間： 1–2 分鐘

※ 過久會變苦，口感清淡更佳

適合搭配的茶點

綠茶適合搭配：

  • 和菓子
  • 抹茶甜點
  • 水果

花茶 Flower Tea

簡介

產地： 中國、台灣

香氣 / 味道： 花香，口感柔和

特色： 通常以綠茶或烏龍茶為底，混合花瓣製作，香氣濃郁。

沖泡建議

水溫： 80–90°C

茶水比例： 1g 茶葉 : 50ml 水

浸泡時間： 2–3 分鐘

適合搭配的茶點

花茶適合搭配：

  • 瑪德蓮
  • 小蛋糕
  • 果乾

奶茶 Milk Tea

簡介

起源： 以紅茶為基底，加入牛奶或奶精調製而成，源自英式飲茶文化，後來在亞洲發展出多樣變化。

香氣 / 味道： 茶香溫潤、奶香柔和，口感圓潤順口。

特色： 奶香能中和茶的澀感，是入門者與甜點搭配的熱門選擇。

沖泡建議

基底茶： 阿薩姆紅茶、錫蘭紅茶、伯爵紅茶

水溫： 90–95°C

比例建議： 茶 7 : 奶 3（可依喜好調整）

小技巧： 先泡濃茶再加熱牛奶，風味較不稀釋

常見類型

  • 英式奶茶： 紅茶＋牛奶，茶味明顯
  • 港式奶茶： 茶味濃厚、口感厚實
  • 台式奶茶： 可加糖、珍珠，變化豐富

適合搭配的茶點

奶茶適合搭配：

  • 司康
  • 英式奶油餅乾
  • 巧克力蛋糕
  • 磅蛋糕

抹茶小蛋糕

簡介

以日本抹茶製作的小蛋糕，香氣濃郁帶苦韻，口感綿密。

特色 / 口感

濕潤軟綿、茶韻清晰、甜度較低。

建議搭配茶種

抹茶、玄米茶、焙茶。

來源 / 背景

源自日式糕點文化，常見於下午茶與手工甜點店。

介紹

抹茶小蛋糕帶有濃郁的茶香，口味偏清爽，利用抹茶本身的苦韻讓甜度更平衡。 是非常適合搭配清爽或烘焙茶類的甜點。

日式抹茶甜點源於茶道文化，強調香氣與層次，是許多茶館的招牌點心。

檸檬蛋糕

簡介

以檸檬皮與檸檬汁調味，酸甜平衡，香氣明亮。

特色 / 口感

外層清香、內部濕潤、入口清爽。

建議搭配茶種

伯爵紅茶、錫蘭紅茶。

來源 / 背景

歐洲家庭茶點中最經典的清新型蛋糕。

介紹

檸檬蛋糕以清新的酸香著稱，是下午茶中非常受歡迎的輕甜點。 香氣明亮、甜度適中，可搭配大多數紅茶。

常見於英式茶會，搭配柑橘風味茶更能凸顯其香氣。

輕乳酪蛋糕

簡介

質地介於慕斯與蛋糕之間，入口即化。

特色 / 口感

輕盈、綿密、奶香柔和。

建議搭配茶種

烏龍茶、茉莉綠茶。

來源 / 背景

源自日本輕乳酪文化，是亞洲茶點常見品項。

介紹

輕乳酪蛋糕口感細緻滑順，比傳統重乳酪更清爽不膩， 與淡雅茶類十分搭配。

起源於日本甜點店，標榜入口即化的輕盈口感。

檸檬塔

簡介

以酥脆塔皮搭配酸甜檸檬凝乳，是清爽度高的法式甜點。

特色 / 口感

塔皮酥脆、內餡滑順、酸度明亮。

建議搭配茶種

伯爵茶、英式紅茶。

來源 / 背景

源自法國傳統甜點，常見於下午茶與咖啡館。

介紹

檸檬塔以其清爽酸甜的檸檬凝乳著稱，塔皮酥香、內餡明亮，是能讓味蕾提神的 清新甜點，非常適合搭配具柑橘香氣的紅茶類。

檸檬塔在法式甜點中地位經典，以其簡單卻突出的風味深受甜點師推崇。

莓果塔

簡介

酥脆塔皮搭配卡士達與新鮮莓果，酸甜兼具的經典法式塔點。

特色 / 口感

酸甜平衡、果香濃郁、卡士達柔滑。

建議搭配茶種

花茶、果香紅茶。

來源 / 背景

法式甜點代表之一，以華麗擺盤和鮮果風味著稱。

介紹

莓果塔以其豐富的新鮮水果與卡士達醬聞名。 酥脆的塔皮與水嫩莓果形成絕佳口感對比，是視覺與味覺兼具的甜點。

在歐洲咖啡館中十分常見，是下午茶套餐的熱門品項。

水果塔

簡介

以多種新鮮水果搭配柔滑卡士達與塔皮，是清爽系塔點代表。

特色 / 口感

水果香甜、卡士達柔滑、整體輕盈。

建議搭配茶種

綠茶、白茶。

來源 / 背景

常見於義式及法式甜點中，是夏季午茶的熱門選擇。

介紹

水果塔以其自然的水果風味和清爽口感受到喜愛。 非常適合搭配清新類茶飲，如綠茶或白茶。

多變的水果組合讓此甜點一年四季都能有不同風味。

司康 Scone

簡介

經典英式下午茶點，外酥內鬆軟，帶有自然奶香。

特色 / 口感

外層微酥、內部紮實鬆軟，與奶油和果醬搭配最佳。

建議搭配茶種

英式紅茶、伯爵紅茶、奶茶類。

來源 / 背景

源於蘇格蘭，是英國下午茶三層點心架上的經典甜點。

介紹

司康以香濃奶油與麵粉烘焙而成，是一款簡單卻迷人的茶點。 外酥內軟的口感讓它成為茶會中不可或缺的角色，通常搭配凝脂奶油與果醬享用。

傳統英式下午茶中，司康位於點心架中層或下層，象徵飽足與溫暖。

英式奶油餅乾

簡介

以高比例奶油製作，口感酥鬆、香氣濃郁。

特色 / 口感

入口即化、奶香厚實、甜度適中。

建議搭配茶種

大吉嶺紅茶、阿薩姆紅茶。

來源 / 背景

源於英國，是下午茶常見的基本餅乾之一。

介紹

英式奶油餅乾以酥鬆的質地著稱，香氣由奶油與糖在烘焙中焦化而來， 是最能代表英式家庭風味的經典小點。

這類餅乾的配方簡單，卻在英式文化中歷久不衰，象徵著日常與溫暖。

馬卡龍 Macaron

簡介

以蛋白霜與杏仁粉製作的法式甜點，外脆內軟、色彩繽紛。

特色 / 口感

外殼酥脆、內餡濃郁、甜度偏高。

建議搭配茶種

無糖紅茶、烏龍茶。

來源 / 背景

起源於法國，被視為法式甜點的代表之一。

介紹

馬卡龍以其精緻外型與多變口味聞名， 外層的薄脆與內餡的濃郁形成強烈反差，是甜度最高的下午茶點之一。

馬卡龍的技術要求高，因此被視為甜點工藝中的精品代表。

羊羹 Yokan

簡介

以紅豆、寒天與砂糖製成的日式傳統甜點，口感細緻滑順。

特色 / 口感

質地緊實柔滑、甜味內斂，適合慢慢品嚐。

建議搭配茶種

綠茶（煎茶、抹茶）、焙茶。

來源 / 背景

源自日本茶道文化，常作為正式茶席的配茶甜點。

介紹

羊羹是日本最具代表性的和菓子之一， 以簡單原料展現出細膩口感與沉穩甜味， 特別適合搭配帶有澀感的綠茶。

在茶道中，羊羹象徵「靜」與「節制」， 常作為正式場合中平衡茶味的重要角色。

麻糬 Mochi

簡介

以糯米製成的日式甜點，口感柔軟富有彈性。

特色 / 口感

Q彈柔軟、甜度可依內餡調整。

建議搭配茶種

綠茶、焙茶、烏龍茶。

來源 / 背景

日本傳統節慶與日常點心中皆常見的和菓子。

介紹

麻糬以其柔軟彈牙的口感深受喜愛， 可搭配紅豆、抹茶或水果等多種內餡， 變化性極高。

由於口感溫和，麻糬非常適合作為清香茶飲的搭配甜點， 不會搶走茶本身的風味。

📜 下午茶的起源

下午茶起源於英國，最早是在19世紀維多利亞時代，由安娜公爵夫人所創立。當時，英國社交文化重視下午茶會，從正午到晚餐之間時間過長，安娜公爵夫人發現自己常感到飢餓，於是在下午四點左右享用一杯茶，搭配三明治與甜點。

隨著時間演變，下午茶逐漸成為上流社會的社交活動，茶會不僅是飲茶，也是一種展示品味和禮儀的場合。茶會的形式越來越精緻，茶品和點心種類豐富，例如加入水果蛋糕、司康、三明治和馬卡龍等。

如今，下午茶文化已傳遍世界各地，高級酒店與咖啡館提供各式下午茶套餐，成為現代人休閒聚會的一種方式。無論是家庭聚會、朋友聚會或商務接待，下午茶都是優雅、放鬆且兼具社交的活動。

此外，下午茶的飲茶時間和茶點搭配也逐漸形成一定規則：紅茶常搭配奶油蛋糕或巧克力點心，綠茶與水果、清淡餅乾最搭，而花茶則適合搭配香甜、精緻的小點心。這些習慣延續至今，使下午茶成為品味與文化兼具的休閒方式。

🎩 下午茶禮儀小知識

下午茶不僅是品嚐美食，更是一種社交場合的禮儀展現。了解基本禮儀能讓你在茶會中表現優雅、得體：

  • 茶具使用：品茶時輕拿茶杯，小口品嚐。加茶時應使用茶壺，而非將茶杯靠近壺口。
  • 坐姿與舉止：保持自然端正姿勢，談話語氣柔和。避免大聲喧嘩或敲擊茶具，給人舒適感。
  • 甜點選擇：選擇與茶搭配的點心，如紅茶配奶油蛋糕或巧克力甜點，綠茶配水果或清淡餅乾，花茶配小蛋糕或馬卡龍。
  • 取食順序：建議先品茶，再搭配點心；若有多層點心架，可依順序由下至上享用點心，從口味清淡到濃郁。
  • 餐桌互動：尊重其他賓客，不搶取食物，避免評論口味差異，保持輕鬆禮貌的談話。
  • 服裝與場合：參加正式下午茶建議穿著得體，例如休閒正裝或簡單優雅服飾，保持整潔乾淨。
  • 使用餐巾：餐巾應輕放於膝上，吃點心時可輕擦嘴角，不宜用手直接觸碰食物。
  • 感謝主人：茶會結束時，向主人表達感謝，保持禮貌告別，展現教養與品味。

掌握以上禮儀，不僅能提升下午茶的整體體驗，也讓自己在社交場合中更具優雅氣質。
